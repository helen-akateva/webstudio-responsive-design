## âœ¨ Features

### Responsive Layout
- **Mobile-first design** with breakpoints for tablet (768px) and desktop (1158px)
- **Responsive images** using `<picture>` element and `srcset` for optimal loading
- **Mobile navigation menu** with hamburger button
- **Flexible grid layouts** for portfolio items

### Interactive Components
- **Modal form** for order/contact requests with validation fields
- **Subscription form** in footer with email input
- **Hover effects** on buttons, links, and portfolio items
- **Smooth transitions** and animations throughout the site
- **Overlay effects** on portfolio cards

### Content Sections
- **Hero section** with call-to-action button
- **Features section** showcasing company values with icons
- **Team section** with member cards and social media links
- **Portfolio section** with project gallery and overlay descriptions
- **Footer** with contact information, social links, and subscription form

### Accessibility
- Semantic HTML5 elements (`header`, `nav`, `main`, `section`, `footer`)
- ARIA labels for interactive elements
- Visually hidden headings for screen readers
- Proper form labels and input associations
